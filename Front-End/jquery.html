What is jQuery?

The jQuery movement is seemily everywhere, but if you still remain cursious what jQuery is, Rick Strahi has an excellent introduction to the library here: An inTrodiction to jQuery.

The jQuery Wikipedia page tells us that the jQuery contains the following features:

  • DOM element selection using the croos-browser open source selector engine Sizzle, a spin out off jQuery project.
  • DOM traversal and modification
  • Events
  • CSS Manipulation
  • Effects and animation
  • Ajax
  • Extensibility
  • Utilities - such as browser version and the each function.
  • JavaScript Plugins
